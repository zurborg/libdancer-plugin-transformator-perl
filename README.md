# NAME

Dancer::Plugin::Transformator - Dancer plugin for Net::NodeTransformator

# VERSION

Version 0.100

# DESCRIPTION	

This plugin provides two methods to interact with [Net::NodeTransformator](https://metacpan.org/pod/Net::NodeTransformator).
